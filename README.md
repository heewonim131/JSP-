## 🛠 개발환경
1. 사용언어
    - Java SE 8
    - JSP
2. 라이브러리/프레임워크
    - Bootstrap 4
    - jQuery 3.6
    - JAF, JavaMail API
3. 통합 개발 환경
    - Eclipse 2020-06
4. 데이터베이스
    - MySQL 8.0
5. 운영체제
    - Windows 10
6. 웹 서버
    - Apache Tomcat 8.5
